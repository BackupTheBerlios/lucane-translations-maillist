Hello,

In order to have a better application menu (no more submenus with one item, no
more overcrawded 'misc' menu), I moved every application to new categories.
To display them in the user language, I have to change the
messages.properties files.

Here are the new categories, with applications they are containing. I need the
translation for each new categorie (not for the apps) :

* Communications 
(QuickMessage, Meeting, SendFile, AudioConf)

* Graphics
(Whiteboard, SlideShow)

* Productivity
(Notes, Todo Lists, Calendar, Shared Folder, Forum)

* Tools
(RSS Reader, JMail)

* Help
(About, Help Browser)


A few one aren't changed so you don't need to translate them. However, i'll
list them too :

* Preferences
(JMail Account, User Profile, Calendar (prefs), Password Changer)

* Invisible
(Reminder, QuickLaunch, MainInterface)

* Administration
(Administration, Kick Users, SMTP Server, JMail Admin, Forums,
Shutdown server, SQL Navigator).


Thank you very much,
Vincent.
