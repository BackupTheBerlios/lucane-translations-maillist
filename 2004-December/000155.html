Hello,

I'm proud to announce Lucane 0.7.4 !
Don't be fooled by the version number, this is actually a major release.

The main new thing is WebConnector, a component allowing you to access 
to Lucane's application through the web.
Other applications have evolved : calendar, todolist, forum, ...
A new Slideshow application has been added.

Please read the changes (including screenshots) here :
http://lucane.org/EN/documentation/whatsnew/0.7.4/

Thanks to everyone invovled,
Vincent.
